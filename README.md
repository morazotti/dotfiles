# dotfiles

Este repositório versiona meus dotfiles (arquivos de configuração) e o material necessário para reproduzir meu ambiente.

## O que tem aqui

- Configurações do shell (ex.: `zsh/.zshrc`)
- Configurações do Guix (ex.: `guix/`)
- Outras configurações de programas usadas no meu ambiente

## Notas

- A ideia é que este repositório seja a “fonte da verdade” das minhas configurações.
- Alguns arquivos podem ser consumidos diretamente (symlink/cópia) por ferramentas como o GNU Guix Home.
